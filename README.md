# k0nsult-governance

The **canon and stewardship kit** of the K0NSULT open commons: the governance charter,
the micro-OSPO/solo-maintainer eligibility model, and the machine-checkable
consultation-response backbone for the EU Open Source Strategy (COM(2026)503).

> **Doctrine:** `claim ≤ proof` · engine hidden · agents-not-people · reputation soulbound
> (not crypto) · Human Override · No Password Custody.

## Contents
| File | What it is |
|---|---|
| `CHARTER.md` | Governance charter — engine/commons boundary, doctrine, eligibility artefacts, fork/exit |
| `CONSULTATION-RESPONSE-DRAFT.md` | EU requirement → K0NSULT artefact → evidence class (DRAFT, operator-gated send) |

## Purpose (COM(2026)503)
Answers needs #23 (accelerator template), #24 (stewardship toolkit), #25-28 (foundations),
#32 (OSPO track), #15/#38-39 (catalogue). Proves a bootstrapped SME can steward reusable
modules credibly without first founding a legal foundation.
